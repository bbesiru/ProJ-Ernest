# Azure Cloud Foundation Project

## Project Overview

This project provides a practical introduction to Microsoft Azure cloud services. The objective was to create and configure an Azure Free Tier environment, explore the Azure Portal, understand cloud governance concepts, deploy a basic cloud resource, and review cost management tools.

## Project Objectives

* Create and configure a Microsoft Azure Free Tier account.
* Explore the Azure Portal and its major services.
* Understand Azure governance through subscriptions and resource groups.
* Learn basic Identity and Access Management (IAM) concepts using Azure RBAC.
* Identify the most appropriate Azure region for resource deployment.
* Deploy a basic Azure resource.
* Explore Azure Cost Management tools.

## Tasks Completed

### 1. Azure Account Registration

A Microsoft Azure Free Tier account was successfully created and verified.

### 2. Azure Portal Exploration

The Azure Portal dashboard, navigation menus, service search functionality, and resource management interfaces were explored.

### 3. Governance Setup

A Resource Group named **DevOps-Lab-RG** was created to organize cloud resources.

### 4. Identity and Access Management

Microsoft Entra ID (formerly Azure Active Directory) was explored, and Azure Role-Based Access Control (RBAC) concepts were reviewed.

### 5. Region Selection

The **West Europe** region was selected because it offers low latency and proximity to the deployment location.

### 6. Resource Deployment

A Storage Account was successfully deployed within the Resource Group.

### 7. Cost Management Review

Azure Cost Management and Billing tools were reviewed, including Cost Analysis and Budget configuration options.
