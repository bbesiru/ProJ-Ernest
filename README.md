Azure Cloud Foundation Project
Project Overview

This project demonstrates the setup and configuration of a Microsoft Azure Free Tier account. The objective was to gain hands-on experience with cloud computing concepts including account creation, portal navigation, resource management, security configuration, and cost monitoring.

Step 1: Azure Account Creation
Visited the Microsoft Azure website.
Selected the Azure Free Tier option.
Signed in using a Microsoft account.

Completed phone number verification.
Added a valid payment method for identity verification.

Step 5: Cost Management

Azure Cost Management and Billing tools were explored to monitor cloud spending and avoid exceeding Free Tier limits.

Creating a Budget
Open Cost Management + Billing.
Select Budgets.
Click Add Budget.
Enter a budget name.
Set an amount (for example, $10).
Select Monthly recurrence.
Configure alert thresholds.
Creating a 75% Cost Alert

Example:

Budget Amount: $10
Alert Threshold: 75%

Azure automatically sends a notification when usage reaches 75% of the configured budget.

Evidence







Step 6: Security Configuration
Multi-Factor Authentication (MFA)

MFA adds an additional layer of security beyond passwords.

Steps to Enable MFA
Open Microsoft Entra ID.
Select Users.
Choose the user account.
Select Authentication Methods.
Enable Multi-Factor Authentication.
Register Microsoft Authenticator or a phone number.

Benefits of MFA:

Protects against stolen passwords.
Reduces unauthorized access.
Improves account security.
